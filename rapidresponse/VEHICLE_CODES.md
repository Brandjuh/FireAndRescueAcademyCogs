## üìù Example Usage

### Using Codes (Fastest!)
```
FT2 BC1 HR1
```
*Sends: 2 Fire Trucks, 1 Battalion Chief, 1 Heavy Rescue*

### Using Full Names
```
2 fire trucks, 1 battalion chief, 1 heavy rescue
```

### Mixed Format
```
FT2, chief 1, heavy rescue 1
```

### Multiple Messages (They Add Up!)
```
FT2 BC1
```
*(wait 2 seconds)*
```
HR1
```
*Total: 2 Fire Trucks, 1 Battalion Chief, 1 Heavy Rescue*

---

## üí° Pro Tips

1. **Use codes for speed** - Much faster to type `FT2 BC1` than full names
2. **Multiple messages add up** - Correct yourself without penalty
3. **Case doesn't matter** - `ft2`, `FT2`, `Ft2` all work
4. **Spaces optional** - `FT2BC1` and `FT2 BC1` both work
5. **Wait 2 seconds** - Between messages to avoid rate limit

## üéØ Common Missions

**Small Fire:**
- Usually: `FT1` or `FT2`

**Structure Fire:**
- Usually: `FT2-3, BC1, PT1`

**Major Fire:**
- Usually: `FT4-6, BC2, PT2, HR1-2, MCV1`

**Hazmat:**
- Usually: `FT3-4, BC2, HM1-2, HR1`

**MVA (Vehicle Accident):**
- Usually: `FT1-2, HR1, PC1-2`
